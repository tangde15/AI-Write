# 🔗 绑定功能使用指南

## 功能说明

现在学生、教师、家长三端都可以主动建立绑定关系，让系统中的角色互相关联。

---

## 🎓 学生端绑定

### 步骤：
1. **登录学生账号**
2. **滚动到右侧底部**，找到"绑定管理"卡片
3. **点击"添加绑定"按钮**
4. **选择绑定类型**：
   - 教师
   - 家长
5. **输入对方的用户名**
6. **点击确定**

### 示例：
```
绑定教师：
- 选择类型：教师
- 输入用户名：teacher01
- 点击确定

绑定家长：
- 选择类型：家长
- 输入用户名：parent01
- 点击确定
```

---

## 👨‍🏫 教师端绑定（即将添加）

教师可以通过用户名主动添加学生。

---

## 👨‍👩‍👧 家长端绑定（即将添加）

家长可以通过用户名主动添加孩子。

## ✅ 绑定成功后

### 学生端
- 可以在"绑定管理"看到已绑定的教师和家长
- 教师显示为蓝色标签 👨‍🏫
- 家长显示为橙色标签 👨‍👩‍👧

### 教师端
- 左侧学生列表会显示已绑定的学生
- 可以查看学生作文并批改

### 家长端
- 左侧会显示已绑定的孩子
- 可以查看孩子作文和成长记录

---

## 🧪 快速测试

### 1. 创建测试账号
```
学生账号：student01 / 123456
教师账号：teacher01 / 123456
家长账号：parent01 / 123456
```

### 2. 建立绑定（学生端操作）
```
1. 用 student01 登录
2. 点击右下方"添加绑定"
3. 绑定教师：teacher01
4. 绑定家长：parent01
```

### 3. 验证绑定
```
1. 退出登录
2. 用 teacher01 登录
3. 左侧应该能看到 student01
4. 可以查看学生作文

同理用 parent01 登录也能看到孩子
```

---

## 📝 注意事项

1. **用户名必须准确**：输入的用户名必须是已注册的用户
2. **角色必须匹配**：
   - 绑定教师时，对方必须是教师角色
   - 绑定家长时，对方必须是家长角色
3. **不能重复绑定**：同一对用户只能绑定一次
4. **双向绑定**：
   - 学生绑定教师 = 教师也绑定了学生
   - 学生绑定家长 = 家长也绑定了孩子
